# HRSignal Brand Guidelines (v1)

## Assets

Canonical files:
- Wordmark: `/public/brand/hrsignal-logo.svg`
- Mark (HS monogram): `/public/brand/hrsignal-mark.svg`
- Dark-safe wordmark: `/public/brand/hrsignal-logo-dark.svg`

Favicons:
- `/public/favicon/favicon.ico`
- `/public/favicon/favicon-16x16.png`
- `/public/favicon/favicon-32x32.png`
- `/public/favicon/apple-touch-icon.png`

## Usage

### Which logo to use
- **Header / navigation:** use the **wordmark** (`hrsignal-logo.svg`).
- **Favicon / app icon / small spaces:** use the **mark** (`hrsignal-mark.svg`).
- **Very dark backgrounds:** use `hrsignal-logo-dark.svg` if the default wordmark ever loses contrast.

### Clearspace
- Maintain clearspace equal to the mark’s corner radius (~1/4 of mark width).
- In UI terms: keep **at least 8–12px** padding around the logo in nav bars and cards.

### Minimum size
- Wordmark minimum height: **24px** (recommended 28–32px).
- Mark minimum size: **20×20px** (recommended 24×24px for UI).

### Color
- Primary accent: `#7441F2` (hover: `#825AE0`).
- Avoid recoloring the gradient mark. If you must, keep it within the HRSignal palette.

### Do / Don’t
- Do keep the logo on flat surfaces with sufficient contrast.
- Don’t place the wordmark directly on noisy gradients.
- Don’t stretch the logo (preserve aspect ratio).
